# 🌐 Global Quant AI (全球多模态量化投研中心)

这是一款基于 **大模型 (LLM) + 多模态视觉 (Vision) + 全球舆情雷达 + 多因子量价模型** 的顶级 A 股量化交易中枢系统。
系统采用“云端 AI 大脑 (VPS) + 移动端极速推送 + 本地自动化交易” 的分布式解耦架构。

## 🎯 核心特性 (Core Features)

* **🖥️ 实时硬件护航 (HUD)**：顶栏悬浮显示 VPS 服务器 CPU、RAM、磁盘使用率及外网连通性 Ping 值，全面掌握节点健康状态。
* **💰 资金面门槛定制**：内置细分价格区间档位，包含【AI 现价穿透防过滤】兜底机制，彻底解决海外 IP 被墙导致的漏单。
* **🌍 全站中英双语 (i18n)**：支持无缝切换 `English` 和 `中文` 界面。
* **📊 深度财务体检 (Deep Financials)**：集成 API 财务核心指标验证 (支持校验 ROE / EPS)，直接拉黑存在退市风险的垃圾股。
* **🛡️ 资产级物理熔断**：内置 API Token 实时监控与大盘熔断锁仓。
