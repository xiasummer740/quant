# Quant Global AI Platform V11.0 🚀

这是一个顶级的事件驱动型量化交易终端。系统通过全自动抓取全球宏观经济数据、机构研报预期以及社交舆情热度，利用大语言模型（如 DeepSeek, GPT-4, Gemini 等）进行多因子共振分析，并自动挖掘出潜力个股。

系统内嵌华尔街级别的 12D AI 透视定价模型，结合腾讯全息盘口数据（市盈率、市值、成交量等基本面数据）与 Python 实时算价，实现“基本面+技术面+情绪面”多维立体风控。

## 🌟 核心特性
- **📰 全息舆情矩阵**：自动抓取并对新闻打标签（突发公告 / 机构研报 / 舆情热度）。
- **🧠 12D 深度透视引擎**：对单只股票进行 12 个维度（宏观、基本面、资金、机构等）的实时 AI 穿透解析，并附带基本面共振价格依据。
- **📊 专业级盘口数据墙**：K 线图内置媲美专业炒股软件的盘口数据（换手率、量比、VWAP筹码峰）。
- **🧭 交易风格双擎**：支持【超短线情绪打板】与【中长线价值潜伏】一键切换，AI 底层逻辑跟随裂变！
- **🤖 跨市场防串台**：严格的 Python 物理级拦截，彻底杜绝主板混入创业板等串台现象。
- **🛡️ 军用级双轨安保**：【主控台】与【系统配置】密码彻底分离，即使离开电脑也不用担心核心 API Key 泄露。

## 🛠️ 小白两步一键部署教程 (1-Click Deployment)

无论您的 VPS 在香港 (HK) 还是美国 (US) 或者任何国家，只需复制以下两行代码在终端执行，即可全自动完成整个平台的搭建！

### 第一步：克隆源码并进入目录
~~~bash
git clone https://github.com/xiasummer740/quant.git /var/www/quant
cd /var/www/quant
~~~

### 第二步：执行一键安装向导
~~~bash
sudo bash install.sh
~~~

> 🎯 **极其智能的部署体验！** 脚本运行后，会提示您输入想要绑定的域名（如 `quant.example.com`，无域名直接按回车即可）。输入后，脚本会自动为您安装 Python 与 Nginx 环境、创建数据库、注册后台服务。最强大的是，它会**自动生成专属自签 SSL 证书，并配置好支持 Cloudflare "完全加密 (Full)" 模式的 Nginx 反向代理**，绝不会与其他项目起冲突！您只需打开浏览器访问即可，全程 0 门槛！

## 🔐 默认账号密码与配置漫游
- 初始密码：主控台与系统配置均为 `admin123`
- **安全提示**：部署成功并进入系统后，请立刻在【⚙️系统配置】的底端修改门禁密码。修改后 `admin123` 将永久失效！
- **全域漫游**：系统所有的敏感配置（如您的 API Key、选股偏好、推送 UID 等）全部安全地储存在服务端的 SQLite 数据库（`data/quant.db`）中，**且绝对不会被推送到 GitHub**。这意味着只要您部署成功一次，以后在任意电脑手机上登录即可，换设备无需重新配置。
