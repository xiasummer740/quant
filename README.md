# 🌐 Global Quant AI (全球多模态量化投研中心)

这是一款基于 **大模型 (LLM) + 多模态视觉 (Vision) + 全球舆情雷达 + 多因子量价模型** 的顶级 A 股量化交易中枢系统。
系统采用“云端 AI 大脑 (VPS) + 移动端极速推送 + 本地自动化交易 (QMT/RPA)” 的分布式解耦架构，为资金体量不同的投资者提供降维打击级的实盘解决方案。

## 🎯 核心特性 (Core Features)

* **🌍 7x24 全球宏观映射**：双总线并发监听财联社国内快讯与新浪 7x24 海外宏观动态，AI 自动完成“海外大事件 -> A股对应受惠板块”的跨市场物理降维映射。
* **🧠 多模态沙盒推演**：支持直接上传券商研报截图、K线图表，调用原生视觉大模型进行底层逻辑拆解与胜率评分。
* **🛡️ 资产级物理熔断**：内置 API Token 实时监控油表，超过阈值立即物理拔网线停机；内置上证指数 MA20 大盘风控，熊市单边下跌自动熔断锁仓。
* **🔬 Pandas 多因子严选**：不仅评估 AI 情绪，更叠加基本面 (PE<50, PB<5) 与高级技术面 (MA20+MACD共振 / KDJ+BOLL极限防追高) 的双重过滤。
* **👥 散户情绪共振**：每 5 分钟静默抓取东方财富/雪球全网热度榜，寻找“游资点火+散户跟风+AI逻辑”的终极共振标的。
* **📱 移动端保姆级辅助**：通过 Telegram / WxPusher 实现毫秒级买单推送，并动态计算提供建仓价、止损位、止盈位。

## ⏱️ 每日 3 分钟标准作业流程 (SOP) - 散户手机辅助流

对于不具备 QMT 接口权限的中小资金投资者（路线一），本系统充当您的“顶级智能投顾”。请严格按照以下机械化纪律执行：

### ☀️ 盘前巡检 (09:00 - 09:15) —— 耗时 1 分钟
1. **查油表**：登录 Web 控制台大屏，查看右上角 `今日 API Token 耗损`，确保额度充裕。
2. **看风向**：查看右侧【🔥 AI 主线热力榜】与【👥 散户情绪监控】，如果在昨晚美股时段（开启全天候模式下）AI 已经提前捕捉到了海外重大映射，记下排名前三的板块，今天这就是全市场的阻力最小方向。

### ⚡ 盘中狩猎 (09:15 - 11:30, 13:00 - 15:00) —— 耗时 1 分钟/次
系统极度严苛，宁可错过绝不做错，一天可能仅推送 1-3 次甚至 0 次。
1. **听信号**：听到手机微信/Telegram“叮咚”响起，立即查看买入简报。
2. **看逻辑**：花 5 秒钟扫一眼 AI 给出的【跨市场推演逻辑】与【胜率】。如果带上了 `[全网共振🔥]` 标签，可加大仓位权重。
3. **挂单执行**：
   * 打开手机同花顺/券商 APP。
   * 输入推送中的股票代码。
   * 按照推送给出的 **“现价”** 或更低价格直接挂限价买单。
4. **铁律风控**：一旦买入成交，**必须立刻**在 APP 中设置【条件单】：
   * 设定**止损价**（系统推送的 `🛑止损参考`，通常是 MA20 或 -5%）。跌破无条件离场，保住本金！
   * 设定**止盈价**（系统推送的 `🎯止盈参考`，通常是 BOLL 上轨或 +5%）。触及分批获利了结，绝不贪婪！

### 🌙 盘后复盘 (15:05 之后) —— 耗时 1 分钟
1. **跑回测**：点击大屏底部的 `▶ 运行回测`，系统将自动拉取过去 7 天所有喊单的 3 日实际最高涨幅、最大回撤与胜负判定。
2. **存报表**：点击 `📥 导出 CSV`，用 Excel 打开，建立您自己的量化交易日志，不断优化自己的微操盘感。

## ⚙️ 快速部署与启动 (Quick Start)

### 1. 云端大脑部署 (Linux Ubuntu 22.04+)
~~~bash
# 激活虚拟环境并安装极客依赖
python3 -m venv venv
source venv/bin/activate
pip install fastapi uvicorn motor akshare pandas pydantic python-dotenv openai zhipuai requests aiohttp google-generativeai anthropic redis pytz python-multipart

# 启动核心后台
uvicorn main:app --host 127.0.0.1 --port 8000 --reload
~~~

### 2. 挂载自动化清理守护进程 (CronJob)
~~~bash
crontab -e
# 添加以下任务，每天凌晨 3 点自动清理冗余数据保护服务器磁盘
0 3 * * * /var/www/quant_system/system_maintenance.sh
~~~
